## 二、系统服务划分
主要服务：  

1. 网关服务（API Gateway）
职责：统一入口、路由分发、认证鉴权、限流熔断
配置：黑白单、限流等

2. 用户服务（User Service）
核心功能：用户注册、登录、个人资料、权限管理

3. 内容服务（Content Service）
核心功能：文章、教程、博客内容的CRUD

4. 问答服务（Q&A Service）
核心功能：技术问答、答案投票、最佳答案采纳

5. 评论服务 (Comment Service)
核心功能：对文章、问答、项目的评论系统

6. 通知服务 (Notification Service)
核心功能：站内消息、系统通知
通知类型：评论回复、文章被赞、问题被回答、关注更新

7. 搜索服务 (Search Service)
核心功能：全文搜索、智能提示、搜索结果排名
搜索范围：文章、问答、用户、项目

8. 文件服务 (File Service)
核心功能：图片/文件上传、CDN分发、缩略图生成
